# المعلومات المدنية للمواطنين 🏛️

نظام إدارة بيانات المواطنين مبني بـ Python + Streamlit + SQLite، يدعم البحث والإضافة والتعديل والحذف مع منع التكرار، وإدارة المختارين والأحياء والأسر.

---

## المميزات

- إدارة بيانات المواطنين (إضافة / تعديل / حذف ناعم / استعادة)
- نظام منع التكرار التلقائي للأسماء والمختارين والأحياء
- إدارة المختارين وربطهم بالأحياء والمناطق
- إدارة الأسر وربط الأفراد
- البحث المتقدم بالاسم والهاتف والعنوان والمهنة
- الفلترة حسب المنطقة والحي والمختار
- تصدير البيانات إلى Excel
- إحصائيات ورسوم بيانية
- واجهة عربية كاملة (RTL)

## التثبيت والتشغيل

### 1. تثبيت Python
حمّل Python 3.10+ من [python.org](https://www.python.org/downloads/) وثبّته مع تفعيل خيار "Add to PATH".

### 2. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 3. تشغيل النظام
```bash
# نقر مزدوج على:
run_ui.bat

# أو من سطر الأوامر:
streamlit run app.py
```

النظام يعمل على: **http://localhost:8501**

## نقل المشروع إلى جهاز آخر

### الطريقة 1: مع البيانات (نسخ كامل)
1. انسخ مجلد المشروع كاملاً بما فيه `data/ali_gharbi.db`
2. ثبّت Python والمكتبات على الجهاز الجديد
3. شغّل `run_ui.bat`

### الطريقة 2: من GitHub (بدون بيانات)
```bash
git clone https://github.com/gmc0011asdf-hash/Citizen-Information.git
cd Citizen-Information
pip install -r requirements.txt
streamlit run app.py
```
ثم انسخ ملف `data/ali_gharbi.db` من الجهاز القديم.
